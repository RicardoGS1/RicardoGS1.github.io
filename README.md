# RicardoGS1.github.io

Sitio personal y portafolio alojado con GitHub Pages.

## Descripción
Este repositorio contiene los archivos estáticos (HTML, CSS, imágenes) que componen mi sitio web personal. El propósito es mostrar proyectos, compartir información personal y servir como punto de contacto.

## Demo
El sitio se despliega automáticamente con GitHub Pages en:
https://ricardogs1.github.io

(El dominio es el del repositorio de usuario/organización: `username.github.io`.)

## Tecnologías
- HTML (100%)
- CSS (opcional)
- Recursos estáticos (imágenes, fuentes, etc.)

## Estructura del repositorio
- `index.html` — página principal.
- Archivos y carpetas estáticas (CSS, images, etc.).
- `app-ads.txt` — configuración de anuncios (si aplica).
- `README.md` — este archivo.

## Ejecutar localmente
Para previsualizar el sitio localmente, puedes:
1. Abrir `index.html` directamente en tu navegador, o
2. Levantar un servidor HTTP sencillo desde la raíz del repositorio:
   - Python 3: `python -m http.server 8000`
   - Luego abrir `http://localhost:8000`

## Cómo editar / desplegar
1. Haz cambios en los archivos HTML/CSS en tu máquina.
2. Haz commit y push a la rama `main` (o la rama por defecto del repo).
3. GitHub Pages detectará los cambios y redeplegará el sitio automáticamente.

## Dominio personalizado
Si quieres usar un dominio propio:
- Añade un archivo `CNAME` con tu dominio en la raíz del repositorio.
- Configura los registros DNS (A o CNAME) según la documentación de GitHub Pages.

## Anuncios / Monetización
Este repositorio incluye `app-ads.txt` para configurar redes publicitarias si decides monetizar el sitio.
